# Stable Matrix Factorization
## Requirements
- Linux with Python >= 3.6
- [PyTorch](https://pytorch.org/) >= 1.1.0
- torchvision >= 0.3.0
- tensorboard >= 1.14.0

## Quick Start
```bash
python main.py --gpu 0
```
You can see more options from
```bash
python main.py -h
```
Result files will be saved in `results/`.
